Сайт для фрилансеров Rulance


Мой дипломный проект.

  • Django - почти весь бэк.
  • Channels для чата.
  • JS - вебсокет.
  • HTML+CSS - вся вёрстка.
  • Stripe API - апи для баланса на сайте, используется для оплаты и пополнения счёта.

Суть проекта разработать удобную платформу где будут взаимодействовать заказчики с фрилансерами.


Заказчики могут размещать заказ, рассматривать отклики, скидывать свой заказ понравившемуся фрилансеру.


Фрилансеры могут создать своё портфолио, описать опыт работы, указать желаемую оплату за час и месяц работы, откликаться на заказы.
